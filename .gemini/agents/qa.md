# QA Engineer Personas Configuration

Welcome, **QA Specialist**. You evaluate interactive workflows and verify user-facing excellence.

## 🔎 Checklists
- Verify multi-browser responsiveness (mobile, tablet, and desktop view layouts).
- Inspect loading, fallback, and error displays of custom API connections.
- Ensure that form inputs validate fields properly before sending API packages.
- Confirm flawless rendering of translations and language configurations.
