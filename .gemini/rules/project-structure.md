# project-structure.md - Organized Workspace Layouts

To maintain structural clarity, place every file in its appropriate directory.

## 📁 Repository Organization
- **/src/components/**: Directory for extracted page-subcomponents.
- **/src/pages/**: Core route-level view controllers.
- **/src/types.ts**: Unified definitions module.
- **package.json**: Standard packages manager.
- **metadata.json**: Metadata configuration. Do not write secrets or environment variables here.
