# system-design.md - Unified Design Configurations & Spacing

Establish professional UI design structures.

## 📐 Design Guidelines
- **Desktop Fluidity**: Define maximum margins and wrap elements inside `max-w-7xl mx-auto`.
- **Rhythm & Padding**: Use deliberate, non-uniform responsive spacings.
- **Dark & Light Integration**: Use deep Slate primary backgrounds and high-contrast foreground details.
- **Authentic Engineering Layouts**: Focus strictly on functional user interfaces, avoiding distracting system simulation text (Anti-AI-Slop).
