# accessibility-checklist.md - Inclusive Usability Verification

Our interfaces must adapt to multiple screen profiles, and remain legible for all users.

- [ ] Font pairings maintain high contrast ratios against light or dark slate backgrounds.
- [ ] Minimal 44px touch targets on mobile view layouts.
- [ ] Logical headings tag hierarchy (`h1`, `h2`, `h3`) followed cleanly.
- [ ] Image elements contain meaningful, localized `alt` description values.
