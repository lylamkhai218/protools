# /build - High-Quality Code Execution Protocol

This protocol governs active engineering work. When executing `/build`, write modular, typed, and resilient code.

## 🏗️ Execution Standards

- **Strict Separation of Concerns**: Keep large data assets (like lists of industrial models) separated from interactive views.
- **Type Honesty**: Explicitly define every Prop interface, State container, and return parameter. Do not use generic `any` values.
- **Hook Isolation**: Prevent infinite re-render loops within React `useEffect` containers by using prime parameter primitives instead of reference objects.
- **Aesthetic First**: Align visual styling directly with Tailwind CSS structures, prioritizing legible typography hierarchies, generous padding margins, and high-contrast color choices.
