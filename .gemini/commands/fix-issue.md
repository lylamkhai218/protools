# /fix-issue - Issue Resolution Protocol

A structured framework for identifying, tracking, and resolving user bugs, UX inconsistencies, or functional requests.

## 🐞 Diagnostics & Remediation

1. **Reproduction Checklist**:
   - Trace the interactive user step to identify where the behavior diverges from expectations.
   - Check developer logs or console records.
2. **Implementation Action Plan**:
   - Use `view_file` to review target implementations.
   - Apply a surgical fix using targeted replacement blocks.
   - Run a quick verify command block to guarantee error resolution.
